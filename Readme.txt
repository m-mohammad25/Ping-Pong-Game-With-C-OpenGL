this is a simple Ping Pong game in C# and OpenGL
===============================================
Keys:
Space : start the game
Enter : pause the game
W     : left paddle up
S     : left paddle down
8 (in Numbers pad) : right Paddle Up
2 (in Numbers pad) : right Paddle down
